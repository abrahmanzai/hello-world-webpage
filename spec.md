# Hello World Webpage - Project Specification

## Project Overview
Create a minimal "Hello World" webpage and deploy it live to the internet using GitHub Pages.

## Requirements

### Content Requirements
1. **Name**: Display "Abdul Hameed Rahmanzai"
2. **Greeting**: Include the message "Hello, World! I'm ready to vibe code."
3. **Styling**: Add at least one CSS style (choosing multiple for better presentation):
   - Background color
   - Font styling
   - Centered text
   - Additional visual enhancements

### Technical Requirements
- Single HTML file (`index.html`)
- Embedded CSS styles (no external stylesheet needed for this simple project)
- Semantic HTML5 structure
- Responsive viewport meta tag

## Design Decisions

### Layout
- Use flexbox for centering content vertically and horizontally
- Create a container div for the main content with visual separation from background

### Color Scheme
- Background: Dark blue (#2c3e50) for contrast
- Container: Light gray (#ecf0f1) for readability
- Heading: Red accent (#e74c3c) for visual interest
- Body text: Dark gray (#34495e) for good readability

### Typography
- Use Arial font family (web-safe, clean, readable)
- Heading size: 2.5em for prominence
- Paragraph size: 1.5em for easy reading

### Visual Enhancements
- Rounded corners on container (15px border-radius)
- Box shadow for depth and visual interest
- Adequate padding (50px) for breathing room

## Deployment Strategy
- Use Git for version control
- Host on GitHub Pages
- Repository name: `hello-world-webpage`
- Live URL: `https://abrahmanzai.github.io/hello-world-webpage/`

## Success Criteria
- [x] Webpage displays my name
- [x] Webpage shows the greeting message
- [x] At least one CSS style applied (multiple styles implemented)
- [x] Code pushed to GitHub repository
- [x] GitHub Pages enabled and site is live
- [x] Site accessible via public URL
